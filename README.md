## About The Project

csvMITRE has been created to help collect a CSV-formatted version of the MITRE ATT&amp;CK Framework (https://attack.mitre.org)
<br><br>


<!-- USAGE EXAMPLES -->
## Usage
`python3 csvMITRE.py [-h] [-v]`
### Example
`python3 csvMITRE.py -v`
